# 💻 Code Explanation Tutor (Claude Skill)

## 📌 프로젝트 개요

본 프로젝트는 Claude를 활용하여 사용자가 입력한 코드를 초보자도 이해할 수 있도록 쉽게 설명해주는 AI 튜터를 구현하는 것을 목표로 한다.
특히 프로그래밍을 처음 배우는 학습자들이 코드의 동작 원리를 직관적으로 이해할 수 있도록 돕는 데 초점을 맞추었다.

## 🎯 개발 목적

* 코드 해석 능력이 부족한 초보 학습자 지원
* 복잡한 코드 설명을 간단한 자연어로 변환
* AI를 활용한 학습 보조 도구 구현

## ⚙️ 주요 기능

* 핵심 내용 요약 기능
* 다양한 코드에 적용 가능 (Python 기준)

## 🧠 동작 원리

본 시스템은 Claude의 자연어 처리 능력을 활용하여 코드의 의미를 분석하고 이를 사람이 이해하기 쉬운 형태로 변환한다.

특히 프롬프트 설계를 통해 다음과 같은 역할을 수행하도록 유도하였다:

* 코딩 튜터 역할 부여
* 출력 형식 고정
* 초보자 친화적 설명 생성

---

## ✍️ 사용한 프롬프트

```text
당신은 초보자를 위한 코딩 튜터입니다.
사용자가 입력한 코드를 아주 쉽게 이해할 수 있도록 설명해주세요.

다음 규칙을 반드시 지켜주세요:

1. 코드의 각 줄이 무엇을 하는지 순서대로 설명하세요.
2. 최대한 쉬운 한국어를 사용하세요.
3. 어려운 용어는 사용하지 말고, 필요하면 간단하게 풀어서 설명하세요.
4. 설명은 길지 않게, 핵심만 간단하게 작성하세요.
5. 마지막에 전체 코드를 한 줄로 요약하세요.

출력 형식은 아래와 같이 작성하세요:

[코드 설명]
- 첫 번째 줄: ...
- 두 번째 줄: ...

[한줄 요약]
...
```

---

## 💡 예시 코드

```python
numbers = [3, 7, 12, 5, 8, 10]

even_sum = 0
odd_sum = 0

for num in numbers:
    if num % 2 == 0:
        even_sum += num
    else:
        odd_sum += num

print("짝수 합:", even_sum)
print("홀수 합:", odd_sum)
```

---

## 📤 실행 결과 예시

[코드 설명]

* numbers 리스트에 여러 숫자를 저장합니다.
* even_sum과 odd_sum 변수를 0으로 초기화합니다.
* 리스트를 하나씩 반복하면서 숫자를 확인합니다.
* 짝수이면 even_sum에 더합니다.
* 홀수이면 odd_sum에 더합니다.
* 최종 결과를 출력합니다.

[한줄 요약]
리스트에서 짝수와 홀수를 나누어 각각의 합을 구하는 코드입니다.
